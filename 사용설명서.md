# 수강생 관리 프로그램 - 사용 설명서

## 📦 프로그램 설치하기

### Mac 사용자

1. `수강생관리프로그램_0.1.0_aarch64.dmg` 파일을 더블클릭하세요
2. 열리는 창에서 `수강생관리프로그램` 아이콘을 `Applications` 폴더로 드래그하세요
3. `Applications` 폴더에서 `수강생관리프로그램`을 찾아 더블클릭하여 실행하세요

**첫 실행 시 보안 경고가 나타날 경우:**
- `수강생관리프로그램.app`을 우클릭 (또는 Control + 클릭)
- "열기" 선택
- 경고창에서 "열기" 버튼 클릭
- 다음부터는 더블클릭으로 바로 실행됩니다

### Windows 사용자

1. `.exe` 파일을 더블클릭하여 실행
2. Windows Defender 경고가 나타날 경우:
   - "추가 정보" 클릭
   - "실행" 버튼 클릭

---

## 💾 데이터 저장 위치

모든 데이터는 프로그램 실행 파일 옆 `data` 폴더에 JSON 파일로 저장됩니다:

```
수강생관리프로그램.app/
└── data/
    ├── students.json      (수강생 정보)
    ├── courses.json       (강좌 정보)
    ├── enrollments.json   (등록 정보)
    └── attendances.json   (출석 정보)
```

**장점:**
- 브라우저 캐시를 삭제해도 데이터가 보존됩니다
- `data` 폴더를 복사하면 백업 완료
- 다른 컴퓨터로 옮길 때 `data` 폴더만 복사하면 됩니다

## ✨ 프로그램 기능 소개

### 1. 대시보드
- 전체 강좌 수, 수강생 수, 총 수익을 한눈에 확인
- 최근 등록된 강좌와 수강생 정보 확인

### 2. 강좌 관리
- 새로운 강좌 추가
- 기존 강좌 정보 수정 및 삭제
- 강좌별 수강생 명단 확인
- 출석 체크

### 3. 수강생 관리
- 수강생 등록
- 수강생 정보 수정 및 삭제
- 수강생별 수강 중인 강좌 확인
- 강좌 신청

### 4. 수익 관리
- 전체 수익 현황 확인
- 결제 상태별 관리 (미납, 부분납부, 완납)
- 결제 금액 추가 및 수정

## 🔧 문제 해결

### 프로그램이 실행되지 않을 때
1. 컴퓨터를 재시작하세요
2. 프로그램을 완전히 삭제하고 다시 설치하세요

### 데이터 백업하기
1. 프로그램을 종료하세요
2. `data` 폴더를 다른 곳에 복사하세요
3. USB나 클라우드에 보관하세요

### 데이터 복원하기
1. 백업한 `data` 폴더를 프로그램 위치에 붙여넣기
2. 프로그램을 다시 실행하세요
