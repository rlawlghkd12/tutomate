  - 포기한 학생도 수강생 목록에 유지 (구분 표시)
  - 일괄 환불 시 "1인당 환불 금액" 명확화 + 과환불 방지
- 납부 모달 디자인 통일 (수강신청 · 강좌관리 · 미납자관리)
  - 빠른 선택 버튼 · 결제수단 아이콘 버튼 · 큰 금액 입력
- 설정 페이지 개선
  - 로그인 이메일 복사 · 체험판 업그레이드 버튼
  - PIN 자리수 표시 · 아이콘 · 간격 정리
- 텍스트 크기 설정이 작은 설명 글자에도 반영
- 기타 버그 수정 및 UI 개선
