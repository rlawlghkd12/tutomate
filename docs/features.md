# TutoMate 기능 정의서

> 앱의 모든 사용자 기능을 정의합니다. 테스트 작성 시 이 문서를 기준으로 커버리지를 판단합니다.
> 마지막 갱신: 2026-03-19

---

## 1. 앱 시작 / 인증

### 1.1 초기화 (자동)
- [ ] 앱 실행 시 `initialize()` 자동 호출
- [ ] 기존 Supabase 세션 복원 (`getSession()`) → 없으면 `signInAnonymously()`
- [ ] `user_organizations` 테이블에서 조직 연결 확인 → 있으면 orgId/plan 복원
- [ ] 조직 연결 없으면 자동으로 `create-trial-org` Edge Function 호출 (버튼 없음)
- [ ] 기기 ID: Electron → `getMachineId()` IPC, 웹 → `crypto.randomUUID()` + localStorage
- [ ] 기기 ID는 SHA-256 해시 처리 후 사용
- [ ] Supabase 미설정(환경변수 없음) 시 로컬 전용 모드 (오류 없음)
- [ ] 인증 완료 전 로딩 스피너 표시
- [ ] 인증 실패 시 에러 메시지 + 재시도

### 1.2 세션 관리
- [ ] `onAuthStateChange` 리스너: 모듈 로드 시 1회 등록
- [ ] 세션 만료(SIGNED_OUT) → `organizationId: null`, `isCloud: false` 초기화
- [ ] 토큰 갱신(SIGNED_IN) → 세션만 갱신, orgId 유지

### 1.3 체험판 제한
- [ ] 강좌 최대 **5개** (`maxCourses: 5`)
- [ ] 강좌당 수강생 최대 **10명** (`maxStudentsPerCourse: 10`)
- [ ] 관리자 탭 접근 불가
- [ ] 제한 초과 시 업그레이드 안내

## 2. 레이아웃 / 네비게이션

- [ ] 메뉴: 대시보드, 강좌 관리, 수강생 관리, 캘린더, 수익 관리, 설정

- [ ] 뷰포트 860px 이하에서 자동 접힘 (collapsed)
- [ ] 헤더 햄버거 버튼으로 수동 접기/펼치기
- [ ] 하단에 체험판/라이선스 상태 표시

### 2.2 헤더
- [ ] 페이지별 제목 + 브레드크럼
- [ ] 알림 벨 아이콘 (읽지 않은 알림 카운트 배지)

### 2.3 글로벌 검색
- [ ] `Cmd+K` (Mac) / `Ctrl+K` (Windows) → 검색 모달 열기
- [ ] `Escape` → 모달 닫기
- [ ] 검색 범위: **전체 / 강좌 / 수강생 / 수강 신청** (설정 검색 없음)
- [ ] 검색어 미입력 시 안내 문구 + 팁 표시
- [ ] 검색 결과 없음: "검색 결과가 없습니다" Empty 표시
- [ ] 결과 하단에 "{count}개의 결과" 표시
- [ ] 결과 항목에 매칭 필드명 표시 (예: "매칭: 이름, 전화번호")
- [ ] 결과 클릭 시 해당 페이지로 이동

### 2.4 오프라인 감지
- [ ] 인터넷 연결 끊김 시 헤더 아래 경고 배너: "인터넷에 연결되어 있지 않습니다"
- [ ] 배너 닫기(X) 클릭 시 세션 내 숨김
- [ ] 재연결 시 배너 자동 제거

### 2.5 라우팅
- [ ] 존재하지 않는 경로 → 대시보드로 리다이렉트

## 3. 대시보드 (`/`)

### 3.1 통계 카드
- [ ] 강좌: 전체 강좌 수, 클릭 시 `/courses` 이동
- [ ] 수강생: 전체 수강생 수, 클릭 시 `/students` 이동
- [ ] 납부: 총 납부 금액 `₩{amount}원`, 클릭 시 `/revenue` 이동
- [ ] 납부율: `{percentage}%` (면제 건 제외 계산)
- [ ] 완납: `{count}건`
- [ ] 미납: `{count}건`

### 3.2 강좌 카드 섹션
- [ ] "전체 강좌 ({count})" 제목
- [ ] 강좌별 카드: 이름, 강사, 강의실, 수강 인원 진행바 `{current}/{max}`
- [ ] 카드 클릭 시 강좌 상세 이동
- [ ] "강좌 등록하기" 버튼 → 강좌 개설 모달

### 3.3 차트
- [ ] 강좌별 수익 막대 차트 (수익 초록 / 예상수익 파랑, 범례 하단)
- [ ] 납부 상태 도넛 차트 (완납/부분납부/미납/면제, 중앙에 퍼센트 표시)

### 3.4 알림 자동 생성
- [ ] 대시보드 진입 시 알림 자동 생성 (하루 1회, localStorage `lastNotificationGeneration` 체크)
- [ ] `payment_overdue`: 등록 후 30일 이상 미납/부분납부 수강생
- [ ] `payment_reminder`: 등록 후 7일/14일/21일째 미완납 수강생
- [ ] enrollments, students, courses 데이터 모두 있을 때만 실행

## 4. 강좌 관리 (`/courses`)

### 4.1 강좌 목록
- [ ] 테이블 컬럼: No., 강좌 이름, 강의실, 강사, 강사 전화번호, 수강료, 수강 인원, 상태, 작업
- [ ] 상태 태그 3종: "모집 중"(초록, 80% 미만) / "마감 임박"(주황, 80%~100%) / "정원 마감"(빨강, 100%)
- [ ] 상태 컬럼 필터링: 모집 중 / 마감 임박 / 정원 마감
- [ ] 강좌 이름 컬럼 정렬 (localeCompare)
- [ ] 수강료 컬럼 정렬
- [ ] 수강 인원 컬럼 정렬
- [ ] 강좌 이름 클릭 시 강좌 상세 이동 (링크)
- [ ] 수강료 `₩{fee.toLocaleString()}` 형식
- [ ] 수강 인원 `{current} / {max}` 형식
- [ ] 작업 버튼: 상세, 수정, 삭제
- [ ] 빈 상태: "등록된 강좌가 없습니다" + 강좌 개설 버튼
- [ ] 모바일(1080px 미만): 작업 버튼 → 드롭다운 메뉴

### 4.2 검색/필터
- [ ] 검색 필드 선택: 전체 / 강좌명 / 강의실 / 강사명 / 전화번호
- [ ] 실시간 검색 필터링
- [ ] 검색 결과 없음: "검색 결과가 없습니다" Empty 표시

### 4.3 강좌 개설
- [ ] "강좌 개설" 버튼 클릭 → 모달
- [ ] **필수 필드: 강좌 이름, 강의실, 강사 이름, 강사 전화번호, 수강료, 최대 인원** (전부 필수)
- [ ] 강사 전화번호 자동 포맷 (`010-1234-5678`)
- [ ] 수강료 프리셋 버튼: 2만원, 3만원, 5만원
- [ ] 최대 인원 프리셋 버튼: 15명, 20명, 25명, 30명, 35명
- [ ] 강좌 일정 설정 체크박스 (선택):
  - [ ] 시작일 (DatePicker)
  - [ ] 수업 요일 (Checkbox.Group) + 빠른 선택: "주중", "주말", "전체"
  - [ ] 시작/종료 시간 (TimePicker) + 빠른 선택: "오전반(9-12시)", "오후반(13-17시)", "저녁반(18-21시)"
  - [ ] 총 수업 회차 (InputNumber)
  - [ ] 일정 활성화 시 시작일, 요일, 시작/종료 시간, 총 회차 모두 필수
- [ ] "생성" 클릭 → 성공 토스트 "강좌가 생성되었습니다."
- [ ] 체험판: 강좌 5개 초과 시 생성 차단

### 4.4 강좌 수정
- [ ] "수정" 클릭 → 기존 데이터 채워진 모달
- [ ] 수정 후 "수정" 클릭 → 성공 토스트

### 4.5 강좌 삭제
- [ ] "삭제" 클릭 → 확인 모달 "강좌를 삭제하시겠습니까?"
- [ ] 수강생 있는 강좌: 별도 경고 모달 "⚠️ 수강생이 있는 강좌입니다!" + 수강생 수 표시
- [ ] 수강생 있는 강좌 삭제 버튼: danger(빨강)
- [ ] 확인 → 강좌 + 관련 수강 등록 + 월별 납부 삭제
- [ ] 성공 토스트 "강좌가 삭제되었습니다."

## 5. 강좌 상세 (`/courses/:id`)

### 5.1 헤더
- [ ] "강좌 목록으로" 뒤로 가기 버튼
- [ ] 강좌 제목 표시
- [ ] 강좌 정보: 강사, 강의실, 수강료, 정원 인라인 표시
- [ ] "내보내기" 버튼 (수강생 없을 때 disabled)

### 5.2 통계 카드
- [ ] 수강생 수: `{enrolled} / {maxStudents}` + 진행바
- [ ] 총 수익: `₩{value}원`
- [ ] 예상 수익: `₩{value}원`
- [ ] 완납률: `{percentage}%`

### 5.3 수강생 관리 탭
- [ ] 테이블: 이름, 전화번호, 납부 현황(태그), 납부 금액, 잔여 금액, 할인, 납부 방법, 납부일자, 등록일, 작업
- [ ] 납부 현황 태그: 미납(빨강), 부분납부(주황), 완납(초록), 면제(보라)
- [ ] 이름 컬럼: 정렬 가능
- [ ] 납부 현황 컬럼: 필터 가능 (미납/부분납부/완납/면제)
- [ ] 납부 금액/잔여 금액/할인/납부일자/등록일 컬럼: 정렬 가능
- [ ] 납부 방법 컬럼: 필터 가능 (현금/카드/계좌이체)
- [ ] 행 선택 → "{count}명 선택됨" + "일괄 납부 처리" + "선택 해제" 버튼
- [ ] "납부 관리" 버튼 → PaymentForm 모달
- [ ] "제거" → Popconfirm: "정말 이 수강생을 제거하시겠습니까?" → 수강 해제 + 토스트

### 5.4 월별 납부 탭
- [ ] 월 선택기 (현재 월 ±6개월)
- [ ] "이번 달" 빠른 이동 버튼
- [ ] 월별 통계: 납부 인원 / 납부 합계 / 예상 합계 / 수납률
- [ ] 이름 컬럼: 정렬, 상태 컬럼: 필터 (납부/미납/면제), 할인 컬럼: 정렬
- [ ] 납부 금액 인라인 편집 (blur/Enter 시 저장)
- [ ] 납부 방법 인라인 변경 (Radio: 현금/카드/이체)
- [ ] 납부일 인라인 변경 (DatePicker)
- [ ] 메모 인라인 입력 (blur 시 저장, 레코드 없으면 자동 생성)
- [ ] "완납" 빠른 버튼: 미납 수강생에게만 표시
- [ ] "전체 완납" 버튼: 해당 월 미납/부분납부 일괄 완납

### 5.5 일괄 납부 처리
- [ ] 모달 상단: 선택 수강생 수, 총 예상 금액, 현재 총 납부액, 총 잔여 금액 요약
- [ ] 납부 방식: "고정 금액" 또는 "비율(%)" 라디오
- [ ] 고정 금액 빠른 버튼: "절반 (₩{half})", "전액 (₩{full})"
- [ ] 비율 빠른 버튼: 25%, 50%, 100%
- [ ] 납부 방법: 미지정 / 현금 / 카드 / 계좌이체
- [ ] 미리보기: 1인당 납부액 + 총 납부액 실시간 표시
- [ ] 적용 방식: 기존 납부액에 추가(+) (덮어쓰기 아님)

### 5.6 내보내기
- [ ] 모달 제목: "수강생 내보내기"
- [ ] 필드 선택: 이름, 전화번호, 이메일, 주소, 생년월일, 납부 현황, 납부 금액, 할인 금액, 잔여 금액, 납부 방법, 납부일자, 등록일, 메모
- [ ] 필드 선택 count: "{선택수}/{전체수}"
- [ ] Excel / CSV 형식
- [ ] 성공 토스트: "Excel/CSV 파일이 다운로드되었습니다."

## 6. 수강생 관리 (`/students`)

### 6.1 수강생 목록
- [ ] 테이블: No., 이름, 전화번호, 강좌(태그), 주소, 메모, 작업
- [ ] 강좌 태그(파란색) 클릭 → 해당 강좌 상세 이동
- [ ] 이름 컬럼: 정렬 가능 (localeCompare)
- [ ] 강좌 컬럼: 필터 가능 (등록 강좌 목록 기준)
- [ ] 작업 버튼: 강좌 신청, 수정 (데스크톱에서 항상 표시)
- [ ] 삭제 버튼: 모바일 드롭다운 메뉴 내에서만 표시 (또는 수정 모달 내 삭제)
- [ ] 모바일(1080px 미만): 작업 → 드롭다운 메뉴 (강좌 신청, 수정, 삭제)
- [ ] 빈 상태 메시지

### 6.2 검색/필터
- [ ] 검색 필드 선택: 전체, 이름, 전화번호, 강좌, 주소, 메모
- [ ] 선택 필드별 placeholder 동적 변경
- [ ] 실시간 검색 필터링

### 6.3 수강생 등록
- [ ] "수강생 등록" 버튼 → 모달
- [ ] 필수 필드: 이름, 전화번호
- [ ] 선택 필드: 생년월일(6자리), 주소, 강좌 신청, 메모
- [ ] 이름 자동완성 (기존 수강생, 최대 8건, 이름+전화번호 표시)
- [ ] 자동완성 선택 → 기존 수강생 정보 + 수강 내역 자동 로드
- [ ] 자동완성 선택 시 info 배너(Alert): "기존 수강생 '{이름}' ({전화번호})의 정보를 수정합니다."
- [ ] 배너 닫기(X) → 자동완성 취소 + 폼 초기화
- [ ] 자동완성 선택 시 토스트: "기존 수강생 '{이름}'님의 정보를 불러왔습니다."
- [ ] 전화번호 자동 포맷 (`010-1234-5678`, 최대 13자)
- [ ] 생년월일 파싱: `630201` → `1963-02-01` (30 이하 → 2000년대)
- [ ] 동일 이름+전화번호 중복 시 경고: "동일한 이름과 전화번호의 수강생이 이미 있습니다."
- [ ] 성공 토스트: "수강생이 등록되었습니다."
- [ ] 신규 등록 후 모달 미닫힘, 이름 필드 포커스 유지 (연속 등록)
- [ ] 체험판: 강좌당 수강생 10명 초과 시 등록 차단

### 6.4 수강생 등록 시 강좌 신청
- [ ] 강좌 Select: 드롭다운 검색, `{name} (₩{fee}) - {count}/{max}명`
- [ ] 이미 선택된 강좌: 취소선 + [선택됨], disabled
- [ ] 정원 마감 강좌: [정원 마감], disabled
- [ ] 선택된 강좌별: 납부 금액, 완납/미납 버튼, 할인, 면제, 납부방법(현금/카드/이체)
- [ ] 강좌 제거 버튼 (DeleteOutlined)
- [ ] 하단 합계: "총 납부: ₩{금액} (할인 ₩{금액}) (면제 {n}건)"

### 6.5 수강생 수정
- [ ] "수정" → 기존 데이터 + 수강 내역 채워진 모달
