# 챗봇 통합 스모크 체크리스트 (수동 검증)

릴리스 전 모든 항목 통과 확인.

## 환경

- [ ] 16GB RAM Mac/Win 빌드
- [ ] 8GB RAM Win 빌드 (성능 게이트 — 응답 10~20초 안내 표시)
- [ ] 4GB Win VM (block 분기 — 챗봇 비활성, 직접 임포트만 노출)

## AI 인프라

- [ ] 첫 진입 → 다운로드 모달 표시 + 사양 진단 카드 노출
- [ ] 다운로드 진행률 % + 누적 GB 표시
- [ ] 다운로드 중단 후 앱 재실행 → 재개 다운로드 (이어받기)
- [ ] sha256 검증 단계 표시 (첫 다운로드 후 sha256 측정·코드 갱신)
- [ ] 다운로드 완료 → 챗봇 사용 가능 (별도 안내 없이 입력창 활성)
- [ ] 설정 > AI 기능 제거 → 모델 삭제 → 다시 다운로드 가능

## 챗봇 — 조회

- [ ] "○○ 학생 결제 언제?" → 정확한 답변 (DB 기반)
- [ ] 존재하지 않는 학생 → "찾을 수 없어요" 답변
- [ ] "이번 달 미납 누구?" → 미납자 목록
- [ ] "○○ 출석률" → 출석/결석 카운트
- [ ] 응답 60초 초과 → 자동 취소 + 안내

## 챗봇 — 임포트

- [ ] 표준 양식 첨부 → preview 카드 → [확정] → result 카드 → DB upsert 확인
- [ ] 미매칭 헤더 1개 포함 → mappingError 카드 + 표준 양식 다운로드 버튼
- [ ] 정규화 실패 행(전화번호 비표준) → 빨강 표시 + 카운트
- [ ] 매핑 캐시: 동일 헤더 두 번째 업로드 → 즉시 매핑 (cacheHit:true)
- [ ] 매핑 캐시: 다른 헤더 집합 업로드 → 새 캐시 저장
- [ ] [취소] 버튼 → 미리보기 카드 닫힘, DB 변경 없음

## 사양 분기

- [ ] 16GB+ → "쾌적" 안내
- [ ] 8GB → "응답 10~20초 걸릴 수 있어요" 안내
- [ ] 4~7GB → warn 분기, 사용자 확인 후 다운로드
- [ ] <4GB → block, 챗봇 비활성, 직접 임포트 폼만 노출

## 직접 임포트 폴백

- [ ] block 사양 PC → /ai-chat 진입 시 직접 임포트 폼만 표시
- [ ] 표준 양식 업로드 → preview 카드 표시
- [ ] 미매칭 헤더 → mappingError 카드 표시
- [ ] 확정은 챗봇 활성 PC에서 처리 안내 (v1)
